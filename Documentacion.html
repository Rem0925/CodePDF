Guía de Uso del Editor de Código

¿Qué es este editor?

Este editor de código permite crear documentos PDF aplicando las normas APA utilizando etiquetas.

Instrucciones de Uso

  1. Escribe tu contenido: Utiliza las etiquetas personalizadas para estructurar tu documento. Por ejemplo, usa @titulo1@ para un título principal.
  2. Inserta contenido: Puedes insertar texto de ejemplo utilizando la etiqueta @Lorem@.
  3. Genera el PDF: Haz clic en el botón "Generar PDF" para crear tu documento en formato PDF.
  4. Visualiza el PDF: El PDF generado se mostrará en un marco de vista previa.

Funcionamiento del Autocompletado

El autocompletado en el editor de código funciona de la siguiente manera:

  1. Detección de Guion: Cuando el usuario escribe un guion (-), el editor activa la función de autocompletado.
  2. Filtrado de Etiquetas: El editor filtra las etiquetas disponibles basándose en lo que el usuario ha escrito después del guion.
  3. Sugerencias: Se muestran sugerencias en un contenedor de autocompletado, permitiendo al usuario seleccionar la etiqueta deseada.
  4. Inserción de Etiquetas: Al hacer clic en una sugerencia, la etiqueta se inserta en el texto.

Ejemplo de Autocompletado

Si el usuario escribe -ti, el editor podría mostrar las siguientes sugerencias:

  • titulo1
  • titulo2
  • subTitulo
  • etc...

Al seleccionar titulo1, el texto se convierte en:

@titulo1@
@/titulo1@

Etiquetas

  • @titulo1-2-3-4-5@ - Título principal y demás niveles de títulos
  • @subTitulo@ - Subtítulo
  • @Introduccion@ - Introducción del documento
  • @Conclusion@ - Conclusión del documento
  • @parr@ - Párrafo de texto
  • @linebreak@ - Salto de línea
  • @pagebreak@ - Salto de página

Notas sobre las Etiquetas de Introducción y Conclusión

Al usar las etiquetas @Introduccion@ y @Conclusion@, puedes incluir el símbolo -| para indicar que el texto siguiente debe ser un nuevo párrafo. Esto permite una mejor organización del contenido y facilita la lectura del documento generado.

Ejemplos de Uso de Etiquetas

Títulos

@titulo1@
Este es el título principal de mi documento.
@/titulo1@

Subtítulo

@subTitulo@
Este es un subtítulo que proporciona más información.
@/subTitulo@

Introducción

@Introduccion@
Este es el párrafo de introducción que describe el tema del documento.
Segundo párrafo de la introducción.
@/Introduccion@

Párrafo

@parr@
Este es un párrafo de texto que contiene información relevante.
@/parr@

Conclusión

@Conclusion@
Este es el párrafo de conclusión que resume los puntos clave.
Otro párrafo de la conclusión.
@/Conclusion@

Salto de Línea

@linebreak@
3
@/linebreak@

Salto de Página

@pagebreak@
@/pagebreak@
