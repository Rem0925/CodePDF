Introducción

¡Bienvenido a la guía de usuario del Generador de Documentos PDF! Esta herramienta te permite crear documentos PDF profesionales a través de un editor de código personalizado con etiquetas específicas.

El generador ha sido diseñado para facilitar la creación de documentos estructurados, informes académicos, manuales y cualquier tipo de documento que requiera un formato consistente y profesional.

Características principales:

Sistema de etiquetas

Utiliza etiquetas personalizadas para definir el formato y la estructura del documento.

Autocompletado

Sistema inteligente que sugiere etiquetas mientras escribes para agilizar el proceso.

Generación instantánea

Convierte tu código en un PDF profesional con un solo clic.

Guardado automático

Tus documentos se guardan automáticamente para que nunca pierdas tu trabajo.

Interfaz de Usuario

La interfaz del generador está dividida en dos áreas principales: el editor de código y la vista previa del PDF.

Componentes principales:

  1. Editor de código

    Área principal donde escribirás tu documento utilizando etiquetas personalizadas. El editor cuenta con resaltado de sintaxis para facilitar la visualización de las etiquetas.

  2. Panel de autocompletado

    Aparece al escribir "-" seguido de texto, mostrando sugerencias de etiquetas disponibles.

  3. Botón de generación

    El botón "Generar Documento" procesa el código y crea el PDF con el formato especificado.

  4. Vista previa

    Área donde se muestra el PDF generado para su revisión inmediata.

  5. Notificaciones

    Mensajes de éxito o error que aparecen en la parte superior para informar sobre el estado del proceso.

Cómo Utilizar

El uso básico del generador de documentos consiste en escribir contenido utilizando etiquetas especiales que definen el formato y la estructura del documento final.

Pasos básicos:

  1. Planifica la estructura de tu documento

    Antes de comenzar, planifica qué secciones necesitará tu documento (portada, índice, introducción, capítulos, conclusión, etc.).

  2. Comienza con una portada

    Utiliza la etiqueta @Portada@ para crear una portada profesional para tu documento. 

    @Portada@

Universidad Autónoma
-|
Título del Documento
-|

-|
Fecha: Mayo 2025

-|
@/Portada@
  3. Agrega un índice

    El índice se genera automáticamente cuando usas la etiqueta @Indice@. Incluirá todas las secciones principales con sus números de página. 

    @Indice@
@/Indice@
  4. Escribe la introducción

    Usa la etiqueta @Introduccion@ para la sección introductoria. 

    @Introduccion@
Este documento tiene como objetivo...
@/Introduccion@
  5. Desarrolla el contenido principal

    Utiliza las etiquetas de títulos y párrafos para organizar el contenido principal. 

    @titulo1@
Capítulo 1: Fundamentos
@/titulo1@

@titulo2@
1.1 Conceptos básicos
@/titulo2@

@parr@
En esta sección analizaremos los conceptos fundamentales...
@/parr@
  6. Finaliza con una conclusión

    Termina tu documento con una conclusión usando la etiqueta @Conclusion@. 

    @Conclusion@
En resumen, este documento ha presentado...
@/Conclusion@
  7. Genera el PDF

    Haz clic en el botón "Generar Documento" para crear el PDF final con todo el formato aplicado.

Etiquetas de Texto

Las etiquetas de texto definen el formato y la jerarquía del contenido en el documento. Cada etiqueta tiene propiedades específicas como tamaño de fuente, estilo y márgenes.

Etiqueta Descripción Propiedades Uso recomendado
@titulo1@ Título principal Tamaño: 12pt, Estilo: Negrita, Margen: Centrado Títulos de capítulos o secciones principales
@subTitulo@ Subtítulo principal Tamaño: 12pt, Estilo: Negrita cursiva, Margen: Centrado Subtítulos de secciones principales
@titulo2@ Título secundario Tamaño: 12pt, Estilo: Negrita, Margen: Izquierda Subsecciones dentro de un capítulo
@titulo3@ Título terciario Tamaño: 12pt, Estilo: Negrita cursiva, Margen: Izquierda Subsecciones menores
@titulo4@ Título cuaternario Tamaño: 12pt, Estilo: Negrita, Margen: Izquierda, Indentado Subsecciones de nivel 4
@titulo5@ Título de quinto nivel Tamaño: 12pt, Estilo: Negrita cursiva, Margen: Izquierda, Indentado Subsecciones de nivel 5
@parr@ Párrafo Tamaño: 12pt, Estilo: Normal, Margen: Izquierda, Indentado Texto principal del documento
Consejo

Mantén una estructura jerárquica coherente en tu documento. Por ejemplo, después de un @titulo1@, usa @titulo2@ para subsecciones, y no saltes directamente a @titulo4@.

Ejemplo de uso:

@titulo1@
Marco Teórico
@/titulo1@

@titulo2@
2.1 Antecedentes
@/titulo2@

@parr@
En esta sección se presentan los antecedentes históricos del tema de estudio.
Este párrafo continúa con más información relevante sobre los antecedentes.
@/parr@

@titulo3@
2.1.1 Estudios previos
@/titulo3@

@parr@
Varios autores han abordado esta temática desde diferentes perspectivas.
@/parr@

Etiquetas de Función

Las etiquetas de función controlan el comportamiento del documento y permiten añadir elementos especiales como saltos de página, texto de ejemplo, o secciones predefinidas.

Etiqueta Descripción Uso
@linebreak@ Inserta uno o más saltos de línea Para añadir espacio vertical entre elementos
@pagebreak@ Inserta un salto de página Para comenzar una nueva página
@Lorem@ Inserta texto Lorem Ipsum aleatorio Para generar contenido de relleno en etapas de diseño
@Indice@ Genera automáticamente un índice del documento Para incluir una tabla de contenido
@Introduccion@ Define la sección de introducción Para la introducción del documento
@Conclusion@ Define la sección de conclusión Para la conclusión del documento
@Portada@ Crea una portada formateada Para la primera página del documento

Autocompletado

El editor cuenta con una función de autocompletado que te ayuda a insertar etiquetas rápidamente mientras escribes.

Cómo usar el autocompletado:

  1. Activa el autocompletado

    Escribe un guion seguido de cualquier letra -t para comenzar a buscar etiquetas que contengan "t".

  2. Navega por las sugerencias

    Aparecerá una lista con las etiquetas que coincidan con lo que has escrito. Puedes seguir escribiendo para filtrar más la lista.

  3. Selecciona una etiqueta

    Haz clic en la etiqueta deseada de la lista.

  4. Inserción automática

    La etiqueta se insertará automáticamente en tu código con las etiquetas de apertura y cierre, excepto en el caso de @Lorem@ que inserta directamente texto de ejemplo.

Consejo

Puedes usar la función de autocompletado incluso en medio de una línea. Simplemente escribe - seguido de las primeras letras de la etiqueta que necesitas.

Ejemplo de uso:

Comenzaré escribiendo una introducción:

-Intro[Aparecerá sugerencia de @Introduccion@]

@Introduccion@
Este documento analiza...
@/Introduccion@

Ahora agregaré un párrafo:

-p[Aparecerá sugerencia de @parr@]

@parr@
Este es el contenido del párrafo.
@/parr@

Generación de PDF

Una vez que hayas escrito tu documento utilizando las etiquetas adecuadas, puedes generar el PDF final con un solo clic.

Proceso de generación:

  1. Revisa tu documento

    Antes de generar el PDF, asegúrate de que todas las etiquetas estén correctamente cerradas y que la estructura sea coherente.

  2. Haz clic en "Generar Documento"

    El botón se encuentra en la parte superior de la interfaz. Al hacer clic, comenzará el proceso de generación.

  3. Espera mientras se procesa

    El botón mostrará un indicador de carga mientras se procesa el documento. Este proceso puede tardar unos segundos dependiendo de la complejidad del documento.

  4. Revisa la vista previa

    El PDF generado se mostrará automáticamente en el panel de vista previa, donde podrás revisarlo antes de descargarlo.

  5. Guarda el PDF

    Puedes guardar el PDF generado utilizando las opciones de tu navegador (generalmente haciendo clic derecho en la vista previa y seleccionando "Guardar como...").

Importante

El generador guarda automáticamente tu código en el almacenamiento local del navegador, pero es recomendable que también guardes una copia del código en tu dispositivo para evitar pérdidas de datos.

Características del PDF generado:

  • Formato A4 estándar
  • Numeración automática de páginas
  • Índice con referencias a números de página
  • Tipografía profesional (Times Roman)
  • Márgenes consistentes en todo el documento
  • Estructura jerárquica clara con diferentes niveles de títulos

Consejos y Trucos

Estos consejos te ayudarán a aprovechar al máximo el generador de documentos y crear PDFs profesionales de manera eficiente.

Planifica la estructura

Antes de comenzar a escribir, esboza la estructura completa de tu documento. Esto te ayudará a mantener una organización coherente y facilitará la navegación para los lectores.

Domina los atajos

Utiliza el autocompletado con - seguido de las primeras letras de la etiqueta para agilizar la escritura. Por ejemplo, -p para párrafos o -t para títulos.

Guarda versiones

Aunque el editor guarda automáticamente tu trabajo en el navegador, es recomendable copiar periódicamente el código en un archivo de texto como respaldo adicional.

Usa el separador -|

Dentro de una etiqueta, puedes usar -| para separar párrafos o secciones sin necesidad de crear nuevas etiquetas, lo que facilita la organización del contenido.

Verifica las etiquetas

Asegúrate de que cada etiqueta de apertura @tag@ tenga su correspondiente etiqueta de cierre @/tag@ para evitar errores en la generación del PDF.

Respeta la jerarquía

Mantén una estructura jerárquica coherente de títulos (título1 > título2 > título3) para que el índice se genere correctamente y el documento sea fácil de navegar.

Consejo profesional

Para documentos extensos, considera dividirlos en secciones lógicas utilizando @pagebreak@ para mejorar la legibilidad. Cada capítulo importante puede comenzar en una nueva página.

Solución de Problemas

Si encuentras problemas al utilizar el generador de documentos, aquí hay algunas soluciones para los inconvenientes más comunes.

Síntoma: Al hacer clic en "Generar Documento", aparece un mensaje de error o el PDF no se genera.

Posibles causas y soluciones:

  • Etiquetas mal cerradas: Verifica que cada etiqueta de apertura tenga su correspondiente etiqueta de cierre.
  • Anidación incorrecta: Asegúrate de no anidar etiquetas de manera incorrecta (por ejemplo, no deberías tener una etiqueta @titulo1@ dentro de un @parr@).
  • Memoria insuficiente: Si el documento es muy extenso, puede que el navegador no tenga suficiente memoria. Intenta dividir el documento en partes más pequeñas.
  • Problemas con el navegador: Actualiza tu navegador a la última versión o prueba con un navegador diferente.

Síntoma: El índice generado no incluye todas las secciones que has creado en el documento.

Posibles causas y soluciones:

  • Etiquetas de título incorrectas: Asegúrate de usar las etiquetas de título adecuadas (@titulo1@, @titulo2@, etc.) para las secciones que quieres incluir en el índice.
  • Posición del índice: La etiqueta @Indice@ debe colocarse después de la portada y antes del contenido principal.
  • Títulos duplicados: Si tienes títulos idénticos, puede haber problemas con la generación del índice. Intenta hacer cada título único.

Síntoma: El texto aparece cortado o mal formateado en el PDF generado.

Posibles causas y soluciones:

  • Caracteres especiales: Algunos caracteres especiales pueden causar problemas. Intenta usar comillas rectas (" ") en lugar de comillas tipográficas (" ").
  • Líneas muy largas: Divide los párrafos muy largos en líneas más cortas para mejorar el formato.
  • Problemas con saltos de línea: Si el texto aparece sin espacios entre párrafos, usa la etiqueta @linebreak@ para añadir espacio vertical.
  • Tablas o estructuras complejas: La herramienta no está diseñada para manejar tablas complejas. Simplifica la estructura si es posible.

Síntoma: Al escribir "-" seguido de texto, no aparece el menú de autocompletado.

Posibles causas y soluciones:

  • Clic fuera del editor: Si has hecho clic fuera del editor, haz clic de nuevo dentro del editor para activarlo.
  • Problema con el foco: Presiona Tab o haz clic en el editor para asegurarte de que tiene el foco.
  • JavaScript desactivado: Asegúrate de que JavaScript está habilitado en tu navegador.
  • Recarga la página: A veces, refrescar la página puede resolver problemas temporales con el autocompletado.

Síntoma: Al regresar al editor, tu documento ha desaparecido o se ha restaurado a una versión anterior.

Posibles causas y soluciones:

  • Almacenamiento local borrado: Si has limpiado los datos de navegación, es posible que se haya borrado el almacenamiento local. Siempre guarda copias de seguridad de tu código.
  • Cambio de navegador: El almacenamiento local es específico de cada navegador. Si cambiaste de navegador, no verás tus datos guardados.
  • Modo incógnito: Si estabas usando modo incógnito o privado, los datos no se guardan permanentemente.
  • Espacio de almacenamiento insuficiente: Verifica que tu dispositivo tiene suficiente espacio de almacenamiento.
Recomendación

Si encuentras un problema que no puedes resolver, intenta simplificar tu documento para identificar la causa. Comienza con un documento básico y ve añadiendo secciones gradualmente hasta identificar qué causa el problema.
